# Head

Inn.ts has a `Head` element which allows you to insert tags within the document header. You can import this from `@innatical/inn.ts/head`.

For example:

```tsx
import Head from "@innatical/inn.ts/head";

const Home = () => {
  return (
    <div>
      <Head>
        <title>Home</title>
      </Head>
    </div>
  );
};

export default Home;
```
